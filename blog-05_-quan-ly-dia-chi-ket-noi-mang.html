1. Lớp InetAddress

Giới Thiệu

Lớp InetAddress trong Java được sử dụng để biểu diễn địa chỉ IP của một máy chủ hoặc máy khách trên mạng. Nó hỗ trợ xử lý cả địa chỉ IPv4 và IPv6.

Các Phương Thức Chính

  1. getByName(String host): Lấy địa chỉ IP từ tên miền hoặc địa chỉ IP cụ thể.

  2. getAllByName(String host): Lấy tất cả các địa chỉ IP liên quan đến một tên miền cụ thể.

  3. getLocalHost(): Lấy thông tin địa chỉ IP của máy tính đang chạy chương trình.

  4. getHostName(): Trả về tên máy chủ của địa chỉ IP.

  5. getHostAddress(): Trả về địa chỉ IP dưới dạng chuỗi.

Ví Dụ

import java.net.*;

public class InetAddressExample {
    public static void main(String[] args) {
        try {
            InetAddress address = InetAddress.getByName("www.example.com");
            System.out.println("Host Name: " + address.getHostName());
            System.out.println("IP Address: " + address.getHostAddress());

            InetAddress localAddress = InetAddress.getLocalHost();
            System.out.println("Local Host Name: " + localAddress.getHostName());
            System.out.println("Local IP Address: " + localAddress.getHostAddress());

        } catch (UnknownHostException e) {
            e.printStackTrace();
        }
    }
}

Chức Năng Chính

  • Truy vấn địa chỉ IP của tên miền hoặc máy chủ.

  • Hỗ trợ xử lý IPv4 và IPv6.

  • Lấy thông tin địa chỉ cục bộ của hệ thống.

2. Lớp URL

Giới Thiệu

Lớp URL (Uniform Resource Locator) đại diện cho một địa chỉ tài nguyên trên Internet như trang web, hình ảnh hoặc tệp tin.

Các Thành Phần Của URL

  1. Protocol: Giao thức sử dụng, ví dụ: HTTP, HTTPS, FTP.

  2. Host: Tên miền hoặc địa chỉ IP của máy chủ.

  3. Port: Cổng kết nối (mặc định là 80 cho HTTP và 443 cho HTTPS).

  4. Path: Đường dẫn đến tài nguyên.

  5. Query: Tham số truyền qua URL.

Các Phương Thức Chính

  • getProtocol(): Trả về giao thức được sử dụng.

  • getHost(): Trả về tên máy chủ.

  • getPort(): Trả về số cổng.

  • getFile(): Trả về đường dẫn và tham số truy vấn.

  • openStream(): Mở kết nối đến tài nguyên và trả về luồng dữ liệu.

Ví Dụ

import java.net.*;

public class URLExample {
    public static void main(String[] args) {
        try {
            URL url = new URL("https://www.example.com/index.html?name=test");
            System.out.println("Protocol: " + url.getProtocol());
            System.out.println("Host: " + url.getHost());
            System.out.println("Port: " + url.getPort());
            System.out.println("Path: " + url.getPath());
            System.out.println("Query: " + url.getQuery());
        } catch (MalformedURLException e) {
            e.printStackTrace();
        }
    }
}

Chức Năng Chính

  • Phân tích cú pháp URL.

  • Truy vấn thông tin chi tiết về URL.

  • Hỗ trợ truy cập và xử lý tài nguyên qua mạng.

3. Lớp URLConnection

Giới Thiệu

Lớp URLConnection cho phép thiết lập kết nối mạng với một URL cụ thể và thực hiện truyền dữ liệu giữa máy khách và máy chủ.

Các Phương Thức Chính

  1. openConnection(): Tạo kết nối tới URL.

  2. connect(): Thiết lập kết nối vật lý.

  3. getInputStream(): Đọc dữ liệu từ máy chủ.

  4. getOutputStream(): Gửi dữ liệu tới máy chủ.

  5. setRequestProperty(String key, String value): Thiết lập tiêu đề yêu cầu HTTP.

Ví Dụ

Đọc Dữ Liệu Từ URL

import java.io.*;
import java.net.*;

public class URLConnectionExample {
    public static void main(String[] args) {
        try {
            URL url = new URL("https://www.example.com");
            URLConnection connection = url.openConnection();
            BufferedReader in = new BufferedReader(new InputStreamReader(connection.getInputStream()));
            String inputLine;
            while ((inputLine = in.readLine()) != null) {
                System.out.println(inputLine);
            }
            in.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Gửi Dữ Liệu Tới Máy Chủ

import java.io.*;
import java.net.*;

public class PostExample {
    public static void main(String[] args) {
        try {
            URL url = new URL("https://www.example.com/api");
            HttpURLConnection connection = (HttpURLConnection) url.openConnection();
            connection.setRequestMethod("POST");
            connection.setDoOutput(true);
            DataOutputStream out = new DataOutputStream(connection.getOutputStream());
            out.writeBytes("param1=value1&param2=value2");
            out.flush();
            out.close();

            BufferedReader in = new BufferedReader(new InputStreamReader(connection.getInputStream()));
            String inputLine;
            while ((inputLine = in.readLine()) != null) {
                System.out.println(inputLine);
            }
            in.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Chức Năng Chính

  • Thiết lập kết nối HTTP/HTTPS.

  • Đọc và ghi dữ liệu từ/đến máy chủ.

  • Hỗ trợ giao tiếp với API và dịch vụ web.

Kết Luận

Lớp InetAddress, URL, và URLConnection cung cấp các công cụ mạnh mẽ để quản lý địa chỉ mạng, xử lý URL và kết nối với máy chủ. Chúng là những thành phần cốt lõi để xây dựng các ứng dụng mạng hiện đại và an toàn.
